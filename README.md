# Handwritten Digit Recognition

Deep Learning CNN that recognises handwritten digits (0-9).

## Dataset
MNIST — 70,000 images (60K train + 10K test), auto-downloaded

## Model
CNN with 2 convolutional blocks | PyTorch

## Tools
Python, PyTorch, NumPy, matplotlib, seaborn, scikit-learn

## Results
- Test Accuracy: 99.46%
- All 10 digits recognised with 99-100% F1 score
- Outputs: accuracy curves, confusion matrix, sample predictions

## How to Run
1. pip install torch torchvision matplotlib seaborn numpy pandas scikit-learn
2. python classifier.py
   (MNIST downloads automatically)
